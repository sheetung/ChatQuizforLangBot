# ChatQuizforLangBot

LangBot 对话式测验插件，当前内置 `sbit` 测试。

## 安装方法

通过LangBot插件市场安装

## 使用方式：

- 私聊机器人发送 `/测试 sbit`
- 按题目回复 `A/B/C/D` 或 `1/2/3/4`
- 回复 `取消` 可中止当前测试

## 扩展方式：

- 将新的测试定义放进 `quizzes/` 目录
- 在题库注册表里增加一个新条目

## 致谢：
